# Trader Performance vs Market Sentiment Analysis

### Executive Summary
This project analyzes the correlation between Bitcoin Market Sentiment (Fear/Greed) and execution data from Hyperliquid.

### Key Insights
1. **Leverage Sensitivity:** High-leverage traders show significantly higher PnL volatility during "Fear" days.
2. **Activity Patterns:** Trade frequency spikes during "Greed" periods, often leading to lower average win rates (over-trading).

### How to Run
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run `analysis.ipynb` to see the full data pipeline and visualizations.
